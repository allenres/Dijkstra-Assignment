## Dijkstra’s Algorithm Assignment

This assignment focuses on implementing Dijkstra’s Algorithm to find the shortest paths in a weighted graph.


## Wave 0: Understand

Before writing any code:

- Review the Edge, Node, and DijkstraAssignment classes
- Understand how each class is used in the graph structure
- Identify how:
  - cities (nodes) are stored
  - roads (edges) are represented
  - distances are calculated
- Note: AI is ok for guidance if you get stuck. AI is allowed for this assignment.

## Wave 1: Implement

Now implement Dijkstra’s Algorithm inside your project. You may structure your solution however you want. If you get stuck, refer to pseudoCode.md for guidance.
